Play

Select the picture that matches the letter on the screen.

Options

Play mode can be customised by selecting only certain letters to be played (for example, those that you know your child is familiar with, or those that you feel they might want to practise), and by setting the number of options displayed to make it more/less challenging.

Audio can also be turned on/off.

Select the icon to access these settings.

Select the icon to play the game full-screen - this helps to prevent the child from inadvertently navigating away from the page. This will also enable a better display on smaller screens.

Learn

View pictures that match the letter on the screen. Select a picture to see the word and hear the pronunciation.

Options

Learn mode can be customised by selecting only certain letters to be viewed (for example, those that you know your child is less familiar with), and by setting the number of pictures displayed.

Audio can also be turned on/off.

Select the icon to access these settings.

Select the icon for full-screen display - this helps to prevent the child from inadvertently navigating away from the page. This will also enable a better display on smaller screens.
