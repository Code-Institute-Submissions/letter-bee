Learn

View pictures that match the letter on the screen. Select a picture to see the word and hear the pronunciation.

Options

Learn mode can be customised by selecting only certain letters to be viewed (for example, those that you know your child is less familiar with), and by setting the number of pictures displayed.

Audio can also be turned on/off.

Select the icon to access these settings. (Or select 'Change settings' from the main page.)

Select the icon for full-screen display - this helps to prevent the child from inadvertently navigating away from the page. This will also enable a better display on smaller screens.
