Options

Play mode can be customised by selecting only certain letters to be played (for example, those that you know your child is familiar with, or those that you feel they might want to practise), and by setting the number of options displayed to make it more/less challenging.

Audio can also be turned on/off.

Select the icon to access these settings. (Or select 'Change settings' from the main page.)

Select the icon to play the game full-screen - this helps to prevent the child from inadvertently navigating away from the page. This will also enable a better display on smaller screens.
